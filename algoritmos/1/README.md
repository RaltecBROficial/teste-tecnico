# Tecnologias utilizadas

- .NET 7;
- C#;
- SQL Server;
- EntityFrameworkCore.

Foi utilizada uma arquitetura em monolíto e code first, algumas das patterns aplicadas foram:
- Tratativa de erros;
- Ioc;
- Pattern de mapeamento;
- DTO's;
- Middlewares.

# Grau de dificuldade

Não tive dificuldade na resolução, apenas levou algum tempo para estruturar, caso o tivesse poderia ter feito cobertura de testes de unidade e integração, além de setup de uma imagem docker.
